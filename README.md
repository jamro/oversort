# OverSort
Let's see how much of over-engineering could be put into a simple app that just sort some data :)

# Prerequisites
- Vagrant
- VirtualBox
- Python3

# How to run?
```
python3 init.py
python3 deploy.py
```

Open web browser and go to `http://192.168.10.2`
